# SideLinker Home

Public website and legal pages for SideLinker.

## GitHub Pages

This repo is served from `docs/` via GitHub Pages.

## Key Pages

- `docs/index.html`: homepage with links to legal and review references
- `docs/privacy.html` and `docs/privacy-zh.html`: privacy policy (EN / ZH)
- `docs/terms.html` and `docs/terms-zh.html`: terms of service (EN / ZH)
- `docs/review-ownership.html` and `docs/review-ownership-zh.html`: ownership statement for App Store review (Guideline 5.2.1)
